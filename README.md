# LLM API Project

Минимальный проект для работы с LLM через API (учебный курс по нейронным сетям)

## Задание
Написать код, который:
- Отправляет запрос в LLM через API
- Получает ответ
- Выводит его в консоль (CLI)

## Установка

1. Клонируйте репозиторий
2. Создайте файл `.env` на основе `.env.example`:
   ```bash
   cp .env.example .env
   ```
3. Добавьте свои API ключи в `.env`
4. Установите зависимости:
   ```bash
   pip3 install -r requirements.txt
   ```

## Запуск

```bash
python3 main.py
```

## Безопасность

⚠️ Файл `.env` с реальными ключами **НЕ коммитится** в Git (защищен через `.gitignore`)
