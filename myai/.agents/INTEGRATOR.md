## Роль
Связываю старое с новым. Подключаю z1-z16 к новому core/. Рефакторю telegram_bot.py постепенно.

## При старте
1. Прочитай `STATE.md` — что уже готово в core/
2. Убедись что Phase 1 (Builder) завершена перед началом работы
3. Начни с самого простого: проверь какие zadanie уже можно подключить

## Мои задачи

### Подключение заданий как MCP tools
- [ ] Изучи z1-z16, определи какие имеют повторно используемую логику
- [ ] Оберни их в `@tool` декораторы в `myai/core/tools.py`
- [ ] Протестируй что agent.py их видит

### Рефакторинг Telegram бота
- [ ] Создай `myai/interfaces/telegram/bot.py`
- [ ] Перенеси handlers из `zadanie1/telegram_bot.py`
- [ ] Замени прямые вызовы API на `from myai.core.agent import run`
- [ ] НЕ УДАЛЯЙ `zadanie1/telegram_bot.py` — он должен остаться рабочим

### Единый конфиг
- [ ] Создай `myai/config.yaml` с настройками
- [ ] Все компоненты читают конфиг через `myai/core/config.py`

## Важные правила
- ⚠️ `zadanie1/telegram_bot.py` НЕ ТРОГАТЬ — только читать
- ⚠️ z1-z16 папки НЕ ИЗМЕНЯТЬ
