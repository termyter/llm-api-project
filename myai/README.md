# myai

Персональный AI ассистент. Строится в рамках курса LLM.

## Структура

```
myai/
  core/              ← бизнес-логика (не зависит от интерфейса)
    llm.py           ← LiteLLM, любая модель
    memory.py        ← история диалогов (SQLite)
    agent.py         ← агентный цикл
    tools.py         ← реестр инструментов (MCP)
  interfaces/
    telegram/        ← Telegram бот
    cli/             ← терминальный интерфейс
    api/             ← FastAPI REST (для Flutter)
  data/              ← базы данных
  config.yaml        ← настройки
  .agents/           ← состояние команды агентов

## Запуск агентной команды

При старте нового сеанса скажи:
"продолжи работу агентов, читай myai/.agents/STATE.md"

## Статус

Смотри: myai/.agents/STATE.md
```

## Задания курса (z1-z16)

Старые задания остаются в `zadanie1-16/` — не трогаем.
Новые задания (z17+) пишем сразу в `myai/core/`.
